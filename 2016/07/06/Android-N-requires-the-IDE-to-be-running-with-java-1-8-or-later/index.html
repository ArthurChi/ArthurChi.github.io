Android N requires the IDE to be running with java 1.8 or later

Posted on 2016-07-06 | Edited on 2016-07-09

环境

MAC OS

问题

所遇到问题题目

解决

网上说了一大堆解决的办法, 但是试了, 没有用, 正确的姿势是

  1. 打开命令行
  2. cd到对应的路径下面 /Applications/Android Studio.app/Contents/
  3. sudo open ./Info.plist
  4. 输入密码
  5. 修改JVMVersion对应的值, 应该看到是1.6*,1.7+, 改为1.8
  6. 关闭, 完成
