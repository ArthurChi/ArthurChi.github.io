Insertion sort

同样是两层循环嵌套, 那么他们的效率有什么差别么? 看这里, 随着排序数目的增加, 选择排序的效果是最优的, 其次是插入排序, 最后是冒泡排序.

Quick Sort

  1. 找到左边比mid大的数
  2. 找到右边比mid大的数
  3. 左右交换
  4. 如此往复, 直到两个指针相遇
  5. 把mid放到合适的位置.
  6. 这时将左边块和右边块进行重复上述操作.
